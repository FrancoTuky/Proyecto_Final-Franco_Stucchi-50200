# Aplicacion web de una tienda de videojuegos con diferentes acciones para administradores y usuarios comunes.

Se puede navegar libremente por la página sin necesidad de iniciar sesión. Pero el usuario, para realizar algunas funciones, será requerido iniciar sesión o en 
caso de no tener uno, registrarse, crear un usuario y contraseña.

Los usuarios comunes registrados pueden realizar las siguientes acciones:
-Visualizar los juegos disponibles
-Visualizar las figuras disponibles
-Buscar juegos
-Buscar figuras
-Escribir reseñas/comentarios en la sección correpondiente
-Editar su perfil y avatar/foto

Los administradores de la página/tienda tienen más funciones, ademas de las ya mencionadas en los usuarios comunes:
-Agregar/editar/eliminar juegos
-Agregar/editar/eliminar figuras
-Visualización de los usuarios registrados en la base de datos
-Dar de alta vendedores/empleados
-Dar de baja vendedores/empleados
-Editar vendedores/empleados
-Visualización de los empleados/vendedores disponibles
